# 玛丽居里博后 - Yelp 研究项目

## 文件结构

本文件夹包含 Yelp 数据集研究项目的所有文件：

### 主要文件

- **literature_review.html** ⭐ [关键文件]
  - 完整的文献综述 HTML 版本
  - 包含 35 篇研究论文的详细分析
  - 9 个总结表格：研究方法、主题、变量等

- **literature_review.md**
  - 文献综述 Markdown 版本
  - 与 HTML 版本内容相同

- **project_context.md**
  - 项目背景和研究上下文
  - AI 助手使用的项目说明文件

- **wos_export_data.txt**
  - Web of Science 导出的文献记录
  - 2160 行，包含完整的论文元数据

- **wos_search_query.txt**
  - Web of Science 搜索查询代码
  - 用于检索相关文献

## 数据集

Yelp 数据集可通过官方网站下载：

**下载地址**: https://business.yelp.com/data/resources/open-dataset/

**数据集大小**: 约 10 GB

数据集包含：
- 商业评论和评分
- 用户生成照片
- 商业信息和属性
- 用户数据和行为

## 使用说明

主要文献综述请查看 `literature_review.html` 文件。

所有文件均可直接在浏览器或文本编辑器中打开。
